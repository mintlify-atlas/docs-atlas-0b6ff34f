## What is Codexa?

Codexa is a cloud-based code editor that combines a professional editing experience with an integrated AI assistant. You write code in a Monaco-powered editor, manage your files through a built-in file explorer, and get AI help without leaving the tab.

The AI assistant operates in two modes:

- **Agent mode** — the AI takes action, writing and modifying code on your behalf
- **Ask mode** — ask questions and get explanations without the AI making changes

Inline autocomplete is also available directly in the editor as you type.

## Key capabilities

- **Monaco editor** — the same editor engine that powers VS Code, with syntax highlighting and autocomplete across all supported languages
- **AI assistant** — Agent and Ask modes for writing, explaining, and editing code
- **Inline autocomplete** — context-aware suggestions as you type
- **File explorer** — create, rename, and organize files within a project
- **Project management** — dashboard to create and manage all your projects
- **37 language and framework templates** — from Python and JavaScript to Flutter, Unity, and more
- **50+ editor themes** — customize the editor appearance to your preference

## Who is Codexa for?

Codexa is built for developers who want AI assistance integrated directly into their editor. Whether you're prototyping a new idea, learning a new language, or working through a complex problem, Codexa's AI assistant is available at every step — without switching between tools.
