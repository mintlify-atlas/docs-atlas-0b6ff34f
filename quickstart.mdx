This guide walks you through creating an account, setting up a project, and using the editor and AI assistant for the first time.

<Steps>
  <Step title="Sign in to Codexa">
    Go to the Codexa app and sign in using your Google account. Click **Sign in with Google** on the login page and authenticate with your Google credentials.

    Once signed in, you land on the dashboard where your projects are listed.
  </Step>

  <Step title="Create a new project">
    On the dashboard, click the **Create Project** button to open the project creation dialog.

    Fill in the following:

    - **Project name** — a name for your project (required)
    - **Framework / Language** — choose from 37 available templates, including JavaScript, TypeScript, Python, React, Next.js, Django, Go, Rust, and more

    Click **Create Project**. Codexa creates the project and opens it in the editor automatically.

    <Note>
      Free accounts are limited to 3 projects. Upgrade to a paid plan to create unlimited projects.
    </Note>
  </Step>

  <Step title="Explore the editor">
    The editor is split into three main areas:

    - **File explorer** (left panel) — browse, create, and organize files in your project
    - **Code editor** (center) — a full Monaco editor with syntax highlighting, inline autocomplete, and multi-file support
    - **AI assistant panel** (right panel) — send messages to the AI and view its responses

    Open a file from the file explorer to start editing. Inline autocomplete activates as you type.
  </Step>

  <Step title="Use the AI assistant">
    The AI assistant panel is available alongside your code. Type a message to describe what you want to build or ask a question about your code.

    <Tip>
      Use **Agent mode** when you want the AI to write or modify code for you directly. Switch to **Ask mode** when you want explanations or answers without the AI making changes to your files.
    </Tip>

    For example, in Agent mode you can type: `Create a REST API endpoint that returns a list of users` and the AI will generate the code and add it to your file. In Ask mode, you can type: `What does this function do?` to get an explanation without any changes.
  </Step>
</Steps>
